## サイト概要
### サイトテーマ
自分の好きな音楽のライブの紹介を投稿したり、閲覧できるコミュニテイサイト

### テーマを選んだ理由テイ
音楽が好きで、他人の好きな音楽やライブによく行く人の知識を共有したいと思ったから。

### ターゲットユーザ
音楽好きな人

### 主な利用シーン
好きなライブの概要を投稿するとき、興味のあるアーティストのライブ情報を知りたいとき
